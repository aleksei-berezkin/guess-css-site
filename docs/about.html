Guess CSS!

HTML and CSS puzzler game

Do you think you know CSS well? Here’s a game to test your knowledge! You are given:

  • A frame containing some rendered fragment

  • Three CSS snippets

  • HTML snippet

Your task is to guess which of three CSS snippets was used to render the fragment! After you made your guess the app shows which one was correct, and updates your score displayed at the top.

Navigate between current and done tasks with arrows to the left and right of the frame.

The game has the following puzzlers:

  • Flexbox

  • Position

  • Selectors

  • Display

  • Grid

If you don’t like them all, use app menu to select only a subset.

Privacy policy
